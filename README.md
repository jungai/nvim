## Installation

> required `neovim`(>= 0.10.0), `fzf`, `ripgrep`, `gnu-sed`

install packages

```bash
brew install neovim fzf ripgrep gnu-sed
```

clone this repo into a `.config` folder

```bash
git clone git@github.com:jungai/nvim.git ~/.config/nvim
```
