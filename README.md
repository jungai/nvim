## Installation

> required `neovim`(>= 0.9), `fzf`, `ripgrep`, `gnu-sed`

install packages

```bash
brew install neovim fzf ripgrep
```

clone this repo into a `.config` folder

```bash
git clone git@github.com:jungai/nvim.git ~/.config/nvim
```
